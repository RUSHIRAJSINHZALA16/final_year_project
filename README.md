# Create Environment
```bash

py -m venv <name_of_env>
py -m venv .venv

```

# Activate Environment
```bash

./.venv/Scripts/Activate.ps1

```

# Deactivate
```bash

deactivate

```

# Install Package
```bash

pip install <package_name>

```

# Install Package from requirement.txt
```bash

pip install -r requirement.txt

```

# Generate Requirement.txt from already installed packages
```bash

pip install -r requirement.txt

```

# Run Flask App
```bash

flask --app main.py run

```
